# Portfólio Acadêmico - Uninter 🎓

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido como trabalho prático para a disciplina de **Aplicações Web e Móveis** do curso de Engenharia de Software da Uninter.

## 💻 Sobre o Projeto

O objetivo principal deste projeto é aplicar os conceitos fundamentais de desenvolvimento front-end na criação de uma página web. O site serve como um cartão de visitas digital, apresentando meu perfil, habilidades técnicas e formas de contato.

🔗 **Acesse o projeto online:** [Meu Portfólio](https://pabloshzz.github.io/trabalho_uninter/)

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da página e conteúdo.
* **CSS3:** Estilização, layout e design responsivo.
## 🚀 Como Visualizar

Como se trata de um site estático já hospedado no GitHub Pages, não é necessária nenhuma instalação ou configuração de ambiente. 

Basta acessar o link do projeto acima diretamente no seu navegador.

Caso queira baixar o código para a sua máquina local:
1. Clone este repositório: `git clone https://github.com/pabloshzz/trabalho_uninter.git`
2. Abra a pasta do projeto no seu editor de código (como o VS Code).
3. Abra o arquivo `index.html` no seu navegador.
